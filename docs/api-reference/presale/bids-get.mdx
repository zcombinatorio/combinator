## Overview

Retrieves all verified contributions (bids) for a presale, including contributor wallets, amounts, transaction signatures, and timestamps. Returns both individual contributions and aggregate totals.

<CodeGroup>

```bash curl
curl -X GET "https://api.zcombinator.io/presale/EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v/bids"
```

```javascript fetch
const tokenAddress = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

const response = await fetch(
  `https://api.zcombinator.io/presale/${tokenAddress}/bids`
);

const bids = await response.json();
```

```python requests
import requests

token_address = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"

response = requests.get(
    f'https://api.zcombinator.io/presale/{token_address}/bids'
)

bids = response.json()
```

</CodeGroup>

## URL Parameters

<ParamField path="tokenAddress" type="string" required>
  The Solana token mint address of the presale
</ParamField>

## Response

<ResponseField name="totalRaised" type="number">
  Total amount raised in the presale (in $ZC tokens, decimal format with 6 decimals)
</ResponseField>

<ResponseField name="totalBids" type="number">
  Total number of verified contribution transactions
</ResponseField>

<ResponseField name="contributions" type="array">
  Array of contribution objects, each containing:
  - `wallet` (string): Contributor's wallet address
  - `amount` (number): Contribution amount in $ZC tokens (decimal format)
  - `transactionSignature` (string): Solana transaction signature
  - `createdAt` (string): ISO 8601 timestamp of when the contribution was recorded
</ResponseField>

## Contribution Format

### $ZC Token Amounts
All amounts are in $ZC tokens with 6 decimals:
- **Stored**: Internally stored as smallest units (6 decimals)
- **Returned**: Converted to decimal format (divided by 1,000,000)
- **Example**: 10000 $ZC tokens (10,000.000000)

<Info>
**Token Standard**: Presale contributions are made exclusively in $ZC tokens. The token mint address for $ZC is `GVvPZpC6ymCoiHzYJ7CWZ8LhVn9tL2AUpRjSAsLh6jZC`.
</Info>

## Contribution Tracking

Each contribution is:
- ✅ **Verified on-chain** before being recorded
- ✅ **Associated with a transaction signature** for proof
- ✅ **Timestamped** for contribution tracking
- ✅ **Deduplicated** to prevent double-counting

## Use Cases

### Display Leaderboard
Show top contributors:
```javascript
const { contributions, totalRaised } = await fetch(
  `/presale/${tokenAddress}/bids`
).then(r => r.json());

// Sort by amount (descending)
const leaderboard = contributions
  .sort((a, b) => b.amount - a.amount)
  .slice(0, 10); // Top 10

leaderboard.forEach((contrib, index) => {
  console.log(`${index + 1}. ${contrib.wallet.slice(0, 8)}... - ${contrib.amount.toLocaleString()} $ZC`);
});
```

### Calculate Contribution Percentage
Show each contributor's share:
```javascript
const { contributions, totalRaised } = await fetch(
  `/presale/${tokenAddress}/bids`
).then(r => r.json());

contributions.forEach(contrib => {
  const percentage = (contrib.amount / totalRaised) * 100;
  console.log(`${contrib.wallet}: ${percentage.toFixed(2)}% of total`);
});
```

### Show Recent Contributions
Display latest contributions:
```javascript
const { contributions } = await fetch(
  `/presale/${tokenAddress}/bids`
).then(r => r.json());

// Sort by date (most recent first)
const recent = contributions
  .sort((a, b) => new Date(b.createdAt) - new Date(a.createdAt))
  .slice(0, 5);

recent.forEach(contrib => {
  const time = new Date(contrib.createdAt).toLocaleTimeString();
  console.log(`${time}: ${contrib.amount} $ZC from ${contrib.wallet.slice(0, 8)}...`);
});
```

### Verify Contribution On-Chain
Check contribution transaction on Solana Explorer:
```javascript
contributions.forEach(contrib => {
  const explorerUrl = `https://explorer.solana.com/tx/${contrib.transactionSignature}`;
  console.log(`Verify: ${explorerUrl}`);
});
```

### Find Wallet's Contributions
Check if a specific wallet contributed:
```javascript
const userWallet = "9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM";

const userContributions = contributions.filter(
  contrib => contrib.wallet === userWallet
);

const totalContributed = userContributions.reduce(
  (sum, contrib) => sum + contrib.amount,
  0
);

console.log(`${userWallet} contributed ${totalContributed} $ZC across ${userContributions.length} transactions`);
```

## Aggregate Statistics

### Total Raised
- Sum of all verified contributions in $ZC tokens
- Updated in real-time as new contributions are verified
- Displayed in decimal format (not smallest units)

### Total Bids
- Count of unique contribution transactions
- Each transaction signature is counted once
- Multiple contributions from the same wallet = multiple bids

## Empty State

If no contributions have been made:

```json
{
  "totalRaised": 0,
  "totalBids": 0,
  "contributions": []
}
```

## Integration Tips

<Tip>
**Real-time Updates**: Poll this endpoint periodically during the presale period to show live contribution updates.
</Tip>

<Tip>
**Transaction Verification**: Use the transaction signatures to provide users with links to Solana Explorer for on-chain verification.
</Tip>

<Warning>
**Amount Display**: The amounts are already in decimal $ZC format. Do NOT divide by 10^6 again or values will be incorrect.
</Warning>

## Contribution Ordering

<Info>
**Default Order**: Contributions are returned in database insertion order (typically chronological). Sort client-side for different orderings (by amount, date, wallet, etc.).
</Info>

## Multiple Contributions

Wallets can contribute multiple times:
- Each contribution creates a separate entry
- Same wallet can appear multiple times in the array
- Aggregate wallet contributions client-side if needed
